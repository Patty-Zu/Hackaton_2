Lista de Elementos HTML5

Elementos Raíz

< !doctype html >  : define el documento como un HTML5
< thtml >  : indica que es documento html y contiene codigo html

Metadaros del documento

< head >  : Representa una colección de metadatos acerca del documento, incluyendo enlaces o definiciones
< title >  : Define el nombre del documento y este se muestra en la barra de titulo del navegador. 
< base >  : Define la URL base para las URLs relativas en la página. 
< link >  : Es usada para enlazar JavaScript y CSS externos con el documento HTML actual.
< meta >  : Define los metadatos que no pueden ser definidos usando otro elemento HTML
< style > : Etiqueta de estilo usada para escribir CSS en línea.

Scripting

< script >  : Define ques es codigo en lenguaje JavaScript.
< noscript >  : Define un contenido alternativo a mostrar cuando el navegador no soporta scripting.

Secciones

< body >  : en el se encuentra todo el contenido del documento html, solo pude haber un body. 
< section >  : Se utiliza para seccionar un documento. 
< nav >  : Define una sección que solamente contiene enlaces de navegación 
< article >  : se usa para definir contenido que puede existir independientemente del contenido del documento
< h1, h2, h3, h4, h5, h6 >  : son cabeceras de distintos niveles siendo h1 el mayo y h6  el menor.
< header > : es la cabecera de una pagina o seccion.puede tener el logotipo o titulo del sitio web 
< footer  > : es el pie de pagina , normalmente contiene mensajes de derecho de autor , direccion o contacto 
< address  > : Define una sección que contiene información de contacto.
